📊 기후 데이터 분석

기온 기울기 0
생물 기울기 0
상관계수 r 0

📈 추세선(기울기) 이해하기

추세선은 데이터의 전체적인 변화 방향을 보여줍니다.

• 기울기 > 0 → 시간이 지날수록 값이 증가
• 기울기 < 0 → 시간이 지날수록 값이 감소

👉 기울기의 절댓값이 클수록 변화 속도가 빠릅니다.

💡 질문: 최근 구간의 기울기는 과거보다 더 가파른가?

🌡️ 기온과 생물 변화 비교

기온과 생물 관측값을 함께 보면 계절 변화의 흐름을 파악할 수 있습니다.

• 기온 상승 + 생물 시기 감소 → 계절이 앞당겨짐
• 기온 변화 없음 → 생물 변화도 작을 가능성

👉 두 그래프의 방향을 반드시 함께 비교하세요.

💡 질문: 두 데이터는 같은 방향으로 움직이는가?

🔗 상관계수 (r) 해석하기

상관계수는 두 데이터의 관계를 숫자로 나타냅니다.

• r ≈ 1 → 함께 증가 (강한 양의 관계)
• r ≈ -1 → 반대로 변화 (강한 음의 관계)
• r ≈ 0 → 관계 없음

👉 |r|이 클수록 관계가 강합니다.

💡 질문: 상관계수가 의미 있는 수준인가?

🔍 구간 확대 분석 방법

전체 데이터뿐 아니라 특정 기간만 따로 분석할 수 있습니다.

• 드래그 → 원하는 구간 확대
• 확대 후 기울기와 r 값 확인

💡 질문: 특정 시기에서 변화가 더 크지 않은가?
